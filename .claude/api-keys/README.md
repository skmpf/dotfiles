# API Keys

Store your API keys here:

```bash
# Anthropic
echo "sk-ant-YOUR-KEY" > ~/.claude/api-keys/anthropic
chmod 600 ~/.claude/api-keys/anthropic

# Z.AI
echo "YOUR-ZAI-KEY" > ~/.claude/api-keys/zai
chmod 600 ~/.claude/api-keys/zai
```

Get keys:
- Anthropic: https://console.anthropic.com/settings/keys
- Z.AI: https://z.ai/manage-apikey/apikey-list
